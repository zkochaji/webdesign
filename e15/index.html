The Program

  1. Open Illustrator
  2. Create file - Common 1366 x 768
  3. Using the rectangle tool create a shape that is 400 x 300
  4. Create another rectangle going right from the top center line of rectangle 1, 280 x 120
  5. Create another rectangle going left from the top center line of rectangle 1, 280 x 120
  6. Using the selection tool select the top right rectangle, then using the pen tool click the top right little square
  7. Using the selection tool select the top left rectangle, then using the pen tool click the top left little square
  8. Using the ellippse tool create a 118 x 118 circle in the center of rectangle 1
  9. Change the color of the circle to the color of your shoes
  10. Change the color of the rectangles to your favorite color
  11. Save the file as a pdf
